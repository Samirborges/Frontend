Trabalhando com listas

Listas ordenadas

Quando vamos fazer uma lista de afaseres e númeramos esses itens
e aparece um imprevisto que obriga a adicionarmos um item na lista
e assim temos que mudar a númeração dos itens posteriores. Porém,
existe uma solução. Usando <ol> faremos uma lista ordenada.
Dentro do ol colocamos o <li> para se referir aos itens

Exemplo:

  1. Meta 01
  2. Meta 02
  3. Meta 03
